# Hospital Analytics — SQL Project

## Overview
SQL-based analysis of a hospital database covering patient flow,
service demand, staff attendance, and patient satisfaction.

## Tables Analyzed
- patients | services_weekly | staff | staff_schedule

## Key SQL Techniques
- Window Functions (ROW_NUMBER, PARTITION BY)
- Joins, Subqueries, CASE Statements
- Aggregations, Date Functions, NULL Handling

## Business Insights
- Identified services with highest refusal rates
- Correlated staff attendance with patient satisfaction
- Found weeks and services with highest unmet demand
